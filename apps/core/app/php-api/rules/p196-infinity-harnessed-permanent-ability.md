# P196 — Infinity (∞) — Harnessed Permanent Grants Abilities

## Abstract
The ∞ (infinity) keyword is found exclusively on Infinity cards from the Final Fantasy Universes Beyond set. "∞ — [Ability]" means "As long as this permanent is harnessed, it has [Ability]." Harnessing is a game action (CR 701.64) that attaches one card to another — a "harnessed" permanent gains benefits from the harness state. The infinity symbol represents the limitless power unlocked when a card achieves its ultimate form or attached state, granting abilities that only function when fully "harnessed" (powered up).

## The Definitive Rules

**CR 702.186a** (verbatim): *"∞ (the mathematical symbol for infinity) is a keyword found on Infinity cards. '∞' is followed by ability text. Together, they represent a static ability."*

**CR 702.186b** (verbatim): *"'∞ — [Ability]' means 'As long as this permanent is harnessed, it has [ability].' See rule 701.64, 'Harness.'"*

## The Pattern

```
INFINITY (∞):
  Static ability: "As long as this permanent is harnessed, it has [Ability]"
  Condition: being "harnessed" — attached to another card via the Harness game action (CR 701.64)
  Effect: grants [Ability] while in the harnessed state

  HARNESS (CR 701.64):
    "To harness" = attach a specific card to another card or game object
    Harnessing is a game action specific to Final Fantasy UB set
    Infinity cards are designed to be harnessed by or to other cards
    The harnessed state activates ∞ abilities

  INFINITY + CONDITIONAL ABILITIES:
    Abilities that only function while harnessed (like equipment only granting while equipped)
    Un-harness: the ∞ ability turns off
    Re-harness: the ∞ ability turns back on
    Multiple ∞ abilities: each activates independently while harnessed

  INFINITY CARDS (Final Fantasy UB):
    Special card type: Infinity cards
    Have ∞ keyword followed by ability text
    Part of the "Infinity" system in Final Fantasy Universes Beyond
    Represent characters reaching their ultimate/infinity form
    Examples: powerful characters with ultimate weapons or transformations

  ∞ + POWER SCALING:
    The ∞ abilities are designed to be powerful (they require specific setup to unlock)
    Without harnessing: the card functions without the ∞ ability
    With harnessing: dramatic power increase
    The harness condition acts as a "charging up" or "transforming" mechanic

  ∞ + GAME TRACKING:
    The game tracks whether a permanent is harnessed (like tracking equipment attachment)
    Multiple permanents can be harnessed simultaneously to different objects
    "Harnessed" is a game state tracked by the attachment rules

  LIMITATIONS:
    ∞ abilities only function on the battlefield (where "permanent" is defined)
    Not in hand, graveyard, or exile — "permanent" means battlefield object
    The conditional "as long as harnessed" means it's a continuous effect, on/off as needed
```

## Definitive Conclusions

- **∞ means "as long as this permanent is harnessed, it has [Ability]"** — conditional ability.
- **Harnessing activates ∞**; un-harnessing deactivates it.
- **Infinity cards are Final Fantasy UB set-specific** — harnessing is a game action from those cards.
- **Powerful abilities** locked behind the harness condition — represents ultimate/powered-up state.

## Canonical Example
**Omnislash (Infinity card with ∞ — "Double strike and deal an additional 3 damage to each creature blocked by this creature"):**
Without harnessing: Omnislash functions normally (base P/T, base abilities if any).
Once harnessed (another card attached per harness rules): it gains double strike + the extra damage.
At max harness state: represents Cloud's ultimate limit break — extremely powerful.

**Example 2 — ∞ Conditional Tracking:**
A creature with ∞ — "Hexproof":
Not harnessed: opponents CAN target it.
Once harnessed: gains hexproof; opponents can't target it.
If opponent finds a way to un-harness it (some effect destroys the harnessed card): hexproof is lost.
The permanent itself survives but loses the ∞-granted hexproof.

## Commonly Confused With
- **P178 (Reconfigure)** — Reconfigure toggles between Equipment being a creature and being attached Equipment. ∞ grants abilities while harnessed (a different attachment mechanic).
- **P185 (Living Metal)** — Living Metal grants artifact creature status based on whose turn it is. ∞ grants abilities based on whether the permanent is harnessed.
- **P175 (Soulbond)** — Soulbond pairs two creatures for shared abilities. ∞ is a conditional ability on a single card requiring the harness game state.
